# OWASP WAF Security Lab

## Project Overview
This project demonstrates web application security testing and mitigation of vulnerabilities using custom Cloudflare WAF rules.

The testing was performed in a controlled environment using OWASP Juice Shop.

## Covered Vulnerabilities
- Security Misconfiguration
- Sensitive Data Exposure
- SQL Injection
- Reflected XSS
- Broken Authentication
- Broken Access Control
- Cryptographic Issues
- Improper Input Validation
- Unvalidated Redirects
- Broken Anti-Automation

## Methodology
Each vulnerability includes:
- Attack Phase
- Defense Phase
- Verification Phase

## Tools Used
- Cloudflare WAF
- Browser DevTools
- HTTP analysis
- OWASP Juice Shop
